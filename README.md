# 🌍 Wallpaper World
### One command. Every screen.
Change every screen you control. Instantly.
<br>
Wallpaper World lets you update the desktop wallpapers of multiple computers
<br>
at the same time using real-time WebSocket communication.

- No manual setup.
- No USB transfers.
- No walking around changing each PC.

Just send one command and every connected screen updates immediately!

## 💡 Example Use Cases
- ☕ Running a café? Turn all monitors into digital menu boards.
- 🏢 Managing an office? Push announcements to every employee screen.
- 🎪 Hosting an event? Synchronize displays across the venue.
- 🖥 Computer lab? Instantly show instructions or alerts.
- 🎮 Gaming center? Update promotions across all PCs at once.

## ⚡ How It Works
Clients connect to your server via WebSocket
- The server sends an image URL
- Clients download the image automatically
- Wallpaper updates instantly
